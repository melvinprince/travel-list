# Travel List

## Overview

**Travel List** is a web application designed to help users efficiently plan and manage their travel packing. It allows users to create, customize, and manage packing lists for various trips, ensuring that no essential items are forgotten.

## Features

- **Create Packing Lists**: Users can create new packing lists tailored to specific trips.
- **Customize Items**: Add, edit, or remove items from the packing list as needed.
- **Responsive Design**: The application is optimized for various devices, providing a seamless user experience.

## Technologies Used

- **Frontend**: React.js
- **Build Tool**: Vite
- **Styling**: CSS

## Installation

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/melvinprince/travel-list.git
   cd travel-list
   ```

2. **Install Dependencies**:

   ```bash
   npm install
   ```

3. **Start the Application**:

   ```bash
   npm run dev
   ```

   The application will run on `http://localhost:3000`.

## Usage

1. **Create a New List**: Click on the "New List" button to start a new packing list.
2. **Add Items**: Use the input field to add items to your list.
3. **Manage Items**: Check off items as you pack them or remove items that are not needed.
